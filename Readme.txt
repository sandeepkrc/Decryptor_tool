RSADecryptor
Overview
RSADecryptor is a Python application that allows users to decrypt files using RSA encryption. This README provides instructions for setting up the environment and creating a standalone executable using PyInstaller.

Prerequisites
Before you begin, ensure you have the following installed:

Python: Version 3.x
PyInstaller: Version 4.x or later
You can install PyInstaller using pip:


git clone https://github.com/yourusername/RSADecryptor.git  
cd RSADecryptor  
Install required dependencies:
Ensure all the required libraries for your application are installed. If you have a requirements.txt file, you can install dependencies with:

dist/RSADecryptor  
Running the Executable
To run the executable, simply navigate to the dist folder and run:

bash
./RSADecryptor  # On Linux or Mac  
RSADecryptor.exe  # On Windows  
Usage
Launch the RSADecryptor executable.
Follow the on-screen prompts to decrypt your files.
